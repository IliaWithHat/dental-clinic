User Service
------------------------------------------------------------------------------------------------------------------------
GET    /{role}          -> найти пользователей по роли
GET    /{role}/{userId} -> найти пользователя
POST   /{role}          -> создать пользователя с разными ролями
POST   /{role}/login    -> аутентификация пользователя
PUT    /{role}/{userId} -> обновить пользователя
DELETE /{role}/{userId} -> удалить пользователя
------------------------------------------------------------------------------------------------------------------------


Time Service
------------------------------------------------------------------------------------------------------------------------
GET    /{role}/{doctorId}/working-time -> найти рабочее время врача
POST   /{role}/{doctorId}/working-time -> установить рабочее время врача
DELETE /{role}/{doctorId}/working-time -> удалить рабочее время врача
------------------------------------------------------------------------------------------------------------------------


Appointment Service
------------------------------------------------------------------------------------------------------------------------
GET    /{role}/{userId}/appointment?from={date}&to={date}&state={state} -> найти записи по {date} и {state} к {role}
POST   /{role}/{doctorId}/appointment                                   -> создать запись к врачу
PUT    /{role}/{doctorId}/appointment/{id}                              -> обновить запись
DELETE /{role}/{doctorId}/appointment/{id}                              -> удалить запись
------------------------------------------------------------------------------------------------------------------------


Review Service
------------------------------------------------------------------------------------------------------------------------
GET    /{role}/{doctorId}/review            -> найти отзывы для врача
POST   /{role}/{doctorId}/review            -> создать отзыв на врача
PUT    /{role}/{doctorId}/review/{reviewId} -> обновить отзыв
DELETE /{role}/{doctorId}/review/{reviewId} -> удалить отзыв
------------------------------------------------------------------------------------------------------------------------


Email Service
------------------------------------------------------------------------------------------------------------------------
sendAppointmentConfirmationEmail() -> отправить письмо с подтверждением записи к врачу
sendAppointmentReminderEmail()     -> отправить письмо с напоминанием о записи за день пациенту
sendReportEmail()                  -> отправить письмо с отчётом владельцу
------------------------------------------------------------------------------------------------------------------------
